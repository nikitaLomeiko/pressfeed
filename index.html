Как это работает?

В рамках работы с профессиональным аккаунтом Pressfeed вы можете выбрать помощь менеджера по сопровождению, или куратора

Кому может потребоваться помощь куратора?

  • Собственнику бизнеса
  • Маркетологу
  • Частному эксперту
  • Другим экспертам, чья профессиональная деятельность напрямую не связана с PR-продвижением

Функции менеджеров

Менеджер по
сопровождению

  • Проведение обучения работе с сервисом всех пользователей компании

  • Своевременное информирование обо всех новостях и акциях сервиса

  • Помощь с настройками сервиса и подписками на запросы

  • Связь с отделом по работе с журналистами – для уточнения деталей по публикации, если дедлайны, указанные в запросе, уже вышли. По запросу пользователя

  • Формирование подборок запросов по запросу пользователя

Стоимость сопровождения (со скидкой)

Без дополнительной стоимости

Куратор

Те же функции, плюс:

  • Ежедневный мониторинг выхода публикаций и соблюдения журналистами дедлайнов

  • Регулярная (1 раз в 1 или 2 недели) связь с клиентом – анализ питчей, консультации по оформлению текста

  • Рекомендации по достижению нужного количества публикаций

  • Помощь с поиском запросов, формирование подборок на регулярной основе

  • Предоставление отчета по итогам месяца: количество публикаций, количество ожидаемых публикаций в следующем месяце
