# pressfeed

## Пет-проект для предосталвения куратора по работе с сервисом Pressfeed

### 1. Как запустить

1.1 скачиваем репозиторий

```
git clone https://github.com/nikitaLomeiko/docker.net.designer.git
```

1.2 Запускаем при помощи Live Server, или открываем index.html в браузере

### 2. Информация о проекте

Сайт написан на нативном JS с HTML и CSS, так-же используется препроцессор scss <br>
Основная задача данного пет-проекта - показать навыки верстки а так-же соблюдения pixel-perfect идеологии <br>
Совпадение с макетом = 96%. Из соображений логики некоторые места с большим количествам текста на сайте - не полностью совпадают с макетом. <br>
На сайте так-же используется логика и элементы украшения добавленные собственной инициативой, такие как: <br>

- AOS для плавной подрузки контента
- Rellax - для паралакс эффекта
- Lenis - плавный и мягкий скроллинг
- Собственно-ручно написанный код для красивой подстветки выделеного текста при наводке

Также был сверстан sidebar, которого не было в макете и меню dropdown-кнопки.
